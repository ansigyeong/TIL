# 분할정복

- 분할(Divide): 해결할 문제를 여러 개의 작은 부분으로 나눔

- 정복(Conquer): 나눈 작은 문제를 각각 해결

- 통합(Combine): (필요하다면) 해결된 해답을 모음



<거듭 제곱(Exponentiation) 알고리즘: O(n)>

```python
def Power(Base, Exponent):
  if Base == 0:
    return 1
  result = 1 # Base^0은 1이므로
  for i in range(Exponent):
    result *= Base
  return result
```
